# Attribution

This project is an independent Flutter/Dart application for Gen 3 Pokemon RNG
workflows.

Some bundled runtime data for Gen 3 encounter tables, personal data, and
localized names was generated from public resources provided by PokeFinder.
Those resources are not vendored in this repository as original source files;
they were transformed into the app-specific JSON data model used at runtime.

GPL-3.0-only.
