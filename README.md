# PokeRNG G3

A Flutter-based Gen 3 RNG tool focused on Pokemon FireRed, LeafGreen, and Emerald.

Priority platforms:

- macOS
- iOS
- Windows
- Android

## Current scope

- Flutter multi-platform project scaffold.
- Gen 3 LCRNG core.
- PID, IV, nature, ability slot, shininess, and Hidden Power helpers.
- Static Method 1 generator.
- Initial static RNG screen.

flutter analyze
